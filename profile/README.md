# CLEAR Lab Open Source

[中文](README-CN.md) | English

CLEAR Lab is building an open-source space for robotics, embodied intelligence, simulation, control, and learning-based robot systems.

This page collects our public research code, teaching resources, reproducible experiments, and selected community collaborations.

## Research and Project Areas

- **Robotics control and learning**
- **Embodied AI and Vision-Language-Action models**
- **Simulation, evaluation, and reproducible experiments**
- **Teaching, tutorials, and research engineering tools**

## Featured Initiative

## Selected Public Repositories

Below are selected public repositories from CLEAR Lab. For the full list, please visit the [Repositories](https://github.com/orgs/clearlab-sustech/repositories) tab.

| Area | Repository | Scope |
| --- | --- | --- |
| Robot learning resources | [Awesome-Legged-Robot-Learning](https://github.com/clearlab-sustech/Awesome-Legged-Robot-Learning) | Curated papers and resources for legged robot learning, reinforcement learning, sim-to-real, and humanoids |
| Wheel-legged locomotion | [Wheel-Legged-Gym](https://github.com/clearlab-sustech/Wheel-Legged-Gym) | Isaac Gym environments and training workflows for wheel-legged robot learning |
| Multi-embodiment locomotion | [multi_loco_isaacgym](https://github.com/clearlab-sustech/multi_loco_isaacgym) | Unified policy training for TRON-1 pointed-foot and wheeled-foot modes using diffusion and reinforcement learning |
| Bipedal control | [Bipedal_MPC](https://github.com/clearlab-sustech/Bipedal_MPC) | Model predictive control examples for TRON 1 in MuJoCo simulation |
| Task-space control | [VF-TSC](https://github.com/clearlab-sustech/VF-TSC) | Adaptive feedback gain methods for floating-base task-space control |
| Tutorials and teaching | [MujocoTutorial](https://github.com/clearlab-sustech/MujocoTutorial) | MuJoCo tutorial materials for simulation learning and practice |
| Course code | [ACR2025](https://github.com/clearlab-sustech/ACR2025) | Advanced Control for Robotics course code repository |
